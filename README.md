## 功能说明：

### 定时限速2.0.0版
#### 2025.6.1 定时限速2.0.0：支持多个接口上的设备共同限速

### 定时限速1.2.2版
#### 2023.7.19 定时限速1.2.2：增加更多日期：工作日和休息日，自定义日期1，2，3中间用逗号分隔;加入MAC地址限速，从此不用担心IPV6和IPV4的限速问题。

### 定时限速1.0版
#### 2022.12.24 定时限速在eqos的加强版，加入定时限制等功能。
